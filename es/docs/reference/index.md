# Referencia

Aquí se encuentra el material de apoyo para la edición, planificación, coordinación de contribuyentes y plantillas de procesos reutilizables de CMS.

## Biblioteca de referencia

- [Integración de CMS](/es/docs/reference/cms-integration/) para la configuración de Pages CMS, flujos de trabajo de edición de campañas, manejo de medios y flujo de publicación.
- [Roadmap](/es/docs/reference/roadmap/) para el trabajo completado, las funciones planificadas y los problemas conocidos.
- [Plantilla de solicitud de extracción](/es/docs/reference/pull-request-template/) para la estructura de relaciones públicas predeterminada utilizada para documentar cambios y probar la cobertura.

## Mejor uso

Venga aquí cuando necesite orientación sobre el proceso, contexto de herramientas editoriales o una referencia rápida mientras coordina el trabajo entre los contribuyentes.
